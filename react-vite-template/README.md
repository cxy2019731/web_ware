## 使用 vite2+react 搭建的开发模板

## 主要依赖

    1.react 17.0.2
    2.vite 2.1.5
    3.antd 4.15.0
    4.concent 2.14.17
    5.cxy-react-i18n 1.30.26
    6.axios 0.21.1
    7.qs 6.10.1
    8.react-router 5.2.0
    9.react-router-dom 5.2.0
    10.less 4.1.1
    11.sass 1.32.8

## 特点
    1.css+less+scss全系module支持,无冲突
    2.全局scss变量,无需引用
    3.移动端postcss配置
    4.自动注入老版本浏览器依赖(非全部)

## 开发

    yarn dev

## 构建

    yarn build
